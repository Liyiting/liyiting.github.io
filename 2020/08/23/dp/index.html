    53. 最大子序和

    输入一个整型数组，数组中的一个或连续多个整数组成一个子数组。求所有子数组的和的最大值。

    要求时间复杂度为O(n)。

    1. 思路：
      1. dp[0] = nums[0]
      2. dp[i] = Math.max(dp[i-1] + nums[i], nums[i])

    执行用时：96 ms, 在所有 JavaScript 提交中击败了43.99%的用户

    392. 判断子序列

    给定字符串 s 和 t ，判断 s 是否为 t 的子序列。
