插入节点

插入节点时主要判断当前节点是否为空，如果为空的话直接将根节点root指向新插入节点，如果不为空则根据新节点大小来判断该添加到左子树还是右子树。

搜索节点

搜索主要分为三种：搜索最大节点，搜索最小节点和搜索指定节点。

删除节点

删除节点有三种情况：

  1. 删除的是叶子节点，直接把父节点指向null
  2. 删除的是只包含一个子节点，原本指向它的节点改为指向这个叶子节点
  3. 删除的节点包含两个叶子节点，此时有两种方案：
    • 查找左子树上的最大值
    • 查找右子树上的最小值

当删除的节点包含开两个叶子节点的时候，需要一个查找子树上最大（小）值的方法。

用找到的最大（小）值创建一个临时节点。将临时节点上的值复制到待删除节点上，然后再删除临时节点。
