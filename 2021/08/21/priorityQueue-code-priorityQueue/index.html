优先队列的实现

回到优先队列最一开始面试官问的问题，如果有很多个数组要合并成一个数组肿么办？

这里的解题方法是利用堆排序构建的优先队列来实现。

具体步骤是：

  1. 将每个数组的最小值push进队列
  2. extract出队列最小值后，将这个最小值来源的数组在insert一个进到队列中
  3. 重复第二步直到队列的size为0
